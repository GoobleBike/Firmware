This folder contains a collection of hardware test for the Gooble Bike 2.0

Test #1: Monitor_plot_1
Detects square wave of speed sensor
print to console for plotter
adjustable sample rate

Test #2: monitor_semiperiodi_2
Detects semiperiods of square wave in mS units

Test #3: monitor_frequenza_3
Detects pulse frequency over an adjustable time base

Test #4: monitor_velocita_4
Maps pulse frequency [imp/s] into speed [km/h]

Test #5: monitor_freno_5
Outputs brake proportionally to speed
